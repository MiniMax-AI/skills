# 事件驱动交易策略

核心逻辑：重大事件改变个股/板块短期供需格局，提前布局可获得超额收益。

## 一、业绩公告事件

### 财报季日历
```
1月底：创业板年报业绩预告
3月底：A股年报截止
4月底：A股一季报截止
8月底：A股中报截止
10月底：A股三季报截止
```

### 超预期信号识别
| 信号 | 强度 | 持续性 |
|------|------|--------|
| 净利润大超预期（>30%）| ⭐⭐⭐⭐⭐ | 3~5天 |
| 营收+净利润双超预期 | ⭐⭐⭐⭐ | 3~5天 |
| 指引上调 | ⭐⭐⭐⭐⭐ | 趋势性 |
| 毛利率扩张 | ⭐⭐⭐ | 持续 |
| 现金流转正 | ⭐⭐⭐ | 中期 |

### 操作要点
```
业绩公告前1~3日埋伏（短线）
公告后：若高开低走→利好出尽；若跳空高开→跟进
关键：区分「预期内」和「超预期」
```

## 二、并购重组事件

| 事件类型 | 爆发力 | 风险 |
|---------|--------|------|
| 借壳上市 | ⭐⭐⭐⭐⭐ | 高（审批风险）|
| 资产注入 | ⭐⭐⭐⭐ | 中 |
| 横向并购 | ⭐⭐⭐ | 中 |
| 战略投资 | ⭐⭐ | 低 |

## 三、政策驱动事件

| 政策类型 | 潜伏方式 | 持有期 |
|---------|---------|--------|
| 行业重磅文件 | 文件发布前1~3天 | 1~3天 |
| 补贴政策 | 公告发布日 | 3~7天 |
| 发展规划 | 预期升温期 | 中线 |

## 四、指数调样事件

```
上证/深证/创业板/科创50指数调整：
调入：被动基金买入 → 尾随买入
调出：被动基金卖出 → 踩踏离场
窗口期：生效前5~10个交易日埋伏
```

## 五、解禁事件

| 解禁类型 | 影响 | 策略 |
|---------|------|------|
| 首发原股东 | 大，通常股价承压 | 回避或做空 |
| 定增机构 | 中，看成本价 | 低于成本可博弈 |
| 股权激励 | 影响小 | 忽略 |

## 操作纪律
```
Step 1：事件确认 → 评估「预期差」
Step 2：测算空间（至少10%）
Step 3：分散持仓（单票≤20%）
Step 4：设置止损（-5%硬止损）
Step 5：事件兑现前离场（买预期卖事实）
```
